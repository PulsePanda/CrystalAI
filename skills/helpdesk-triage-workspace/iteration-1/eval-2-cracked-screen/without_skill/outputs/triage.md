## Diagnosis

- **Incident type:** Accidental physical damage (drop). Student Chromebook screen is cracked but device boots.
- **Likely damage:** LCD panel and/or digitizer cracked. If it still turns on and the display is partially functional, the main board is likely intact.
- **Risk:** Display degradation may worsen over time. Cracked glass is a safety concern (sharp edges).
- **Data concern:** None — Chromebooks use cloud storage (Google Drive); no local data at risk.
- **Repair options:**
  1. **Spare device swap (recommended immediate action):** Issue a loaner/spare Chromebook from inventory so the student is back in class immediately. Pull the damaged unit for repair triage.
  2. **Screen repair:** If GIS manages its own repair stock, a Chromebook screen replacement is typically $40–80 in parts. Most models (HP, Lenovo, Dell, Acer school Chromebooks) have readily available screens on iFixit or Amazon. Labor is straightforward for a tech with experience.
  3. **Warranty / ADP check:** Check whether this device is covered under Accidental Damage Protection (ADP) — many school Chromebook purchases include it via Google for Education partners (CDW, SHI, etc.). If ADP is active, file a claim rather than paying out of pocket.
  4. **Insurance / damage policy:** GIS likely has a student device damage policy (co-pay or full replacement billed to family). Front office should check the policy and document the incident.

## Recommended Next Steps

1. Swap to a spare Chromebook — get student back to class.
2. Tag and set aside the broken unit.
3. Check ADP/warranty status on the device (pull serial number from device back or admin console).
4. If ADP covered: file claim. If not: assess repair vs. replace based on device age.
5. Document the incident per GIS device damage policy; notify the classroom teacher and potentially the family per policy.
6. Tech notes: verify the device serial in Google Admin Console to confirm it's enrolled and note condition history.
