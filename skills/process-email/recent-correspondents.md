## Refresh Mechanism

1. Query each Gmail account: `messages list` with `q: "in:sent after:YYYY/MM/DD"` (6 months ago or since last-refreshed)
2. Extract unique recipient addresses from results
3. Merge with existing table, update `Last Sent` dates
4. Remove entries where `Last Sent` is > 6 months ago
5. Skip anyone in the Manually Removed table
6. Update `last-refreshed` in frontmatter
